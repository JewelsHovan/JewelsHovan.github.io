# julienhovan.com

My personal corner of the web: projects, experiments, diagrams, and field notes about AI, software, data, and the things I am still figuring out.

Visit [julienhovan.com](https://julienhovan.com).

## Run locally

```bash
npm install
npm start
```

## Checks

```bash
npm test -- --watchAll=false
npm run build
```

The site is built with React and published through GitHub Pages.
